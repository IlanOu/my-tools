Fab est une bibliothèque de composants UI développée par Epic Games, les créateurs d'Unreal Engine. Elle est conçue pour être utilisée avec Unreal Engine, mais peut également être utilisée dans d'autres projets (Unity, Godot...).

### Fonctionnalités

*   Composants UI prêts à l'emploi : boutons, menus, barres de progression, etc.
*   Personnalisation facile grâce à des styles et des thèmes
*   Support pour les animations et les effets visuels
*   Intégration avec Unreal Engine pour une utilisation transparente

### Avantages

*   Facile à utiliser : les composants sont prêts à l'emploi et peuvent être facilement personnalisés.
*   Performant : Fab est optimisé pour les performances, ce qui signifie que tes UI seront fluides et réactives.
*   Flexible : tu peux utiliser Fab dans différents types de projets, pas seulement les jeux vidéo.
